Data Structures Visualizer

A web-based interactive tool to visualize and explore basic computer science data structures, designed for students, learners, and professionals looking to understand and experiment with arrays, stacks, queues, linked lists, and binary trees.

Features

Visual Representation of Data Structures

Array, Stack (LIFO), Queue (FIFO), Linked List, Binary Tree

Node visualization with arrows for linked lists and hierarchical layout for trees

Smooth fade-in animations on node insertion

Interactive Operations

Insert and remove elements dynamically

Choose which data structure to interact with via a dropdown

Definition Panel

Toggleable sidebar on the right with detailed information about each data structure

Includes description, code example, and illustrative image

Themes

Switch between Default, Dark, Hacker, and Pastel themes for better visualization

Personalization with live theme changes

Personal Description Panel

Fixed panel at bottom-left with a short introduction of the user

Designed for students networking or showcasing career goals

Installation & Usage

Clone or download the repository:

git clone <repository-url>
cd data-structures-visualizer


Open the index.html file in a modern web browser (Chrome, Firefox, Edge, Safari).

Use the controls at the top to:

Select a data structure

Insert or remove elements

Toggle the definition panel

Switch between themes

Example Usage

Insert elements into a stack and watch them appear in LIFO order.

Toggle the definition panel to see the description, example code, and image for each data structure.

Change themes to visualize your structures in Dark, Hacker, or Pastel mode.
