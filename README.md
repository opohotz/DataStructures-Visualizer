# Data Structures Visualizer

Interactive web app to **visualize and explore fundamental computer science data structures**. Perfect for students, learners, and professionals aiming to understand arrays, stacks, queues, linked lists, and binary trees.

---

##  Features

- **Dynamic Visualization**
  - Insert and remove elements in real-time  
  - Arrows for linked lists, hierarchical layout for binary trees  
  - Smooth fade-in animations for nodes  

- **Definition Panel**
  - Toggleable right sidebar with detailed info  
  - Includes description, example code, and illustrative images  

- **Themes**
  - Switch between **Default**, **Dark**, **Hacker**, and **Pastel** modes
